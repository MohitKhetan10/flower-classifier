# flower-classifier
ResNet152 vs VGG19 flower classification notebooks

# 🌸 Flower Classifier Project

This repository contains three Google Colab notebooks:
- `resnet_training.ipynb` → Training ResNet152 on TF Flowers dataset
- `vgg19_training.ipynb` → Training VGG19 on TF Flowers dataset
- `comparison.ipynb` → Accuracy comparison and confusion matrix analysis

## How to Use
1. Open any notebook directly in Google Colab:

2. Run the cells to reproduce results.
3. Dependencies: TensorFlow, Keras, NumPy, Matplotlib, Pillow.
